# mac-setup

- Install Homebrew
- Install git
- Install git clone this repository

```sh
$ sh initSettings/initDownload.sh
```

- default settings

```sh
$ sh initSettings/defaultsSettings.sh
```

- Brew bundle

```sh
$ initSettings/installPackages.sh
```

- Setting dotfiles

```sh
$ initSettings/settingDotfiles.sh
```

- ssh key
```sh
$ devSettings/sshKey.sh
```
